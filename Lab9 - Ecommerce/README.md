# 🛒 ShopUp — Laravel E-Commerce Platform

**ShopUp** is a full-featured e-commerce web application built with **Laravel 13**. It covers the complete shopping journey — from browsing a curated product catalog to placing and tracking orders — and ships with a separate admin area for managing the store.

## ✨ Key Features

### 🧑‍💼 Storefront (Customer)
* **Home page** with featured products and category highlights.
* **Product catalog** with category filtering and clean, slug-based URLs.
* **Product detail pages** with descriptions, pricing, and stock indicators.
* **Shopping cart** that works for both **guests** and **authenticated users** — add, update quantity, and remove items.
* **Checkout flow** for placing orders with order confirmation.
* **Order history** — view past orders and detailed order summaries.
* **Account dashboard & profile management** powered by Laravel Breeze.

### 🔐 Authentication
* Registration, login, password reset, and email verification (Laravel Breeze).
* Middleware-protected customer area.

### ⚙️ Admin Panel (`/admin`)
* **Products CRUD** — create, edit, and delete catalog items with image uploads.
* **Categories CRUD** — organize the catalog.
* **Order management** — view all orders, inspect details, and update statuses.
* **Admin dashboard** with quick metrics and recent activity.

---

## 🛠️ Tech Stack
* **Backend:** PHP 8.3+ & Laravel 13
* **Frontend:** Blade Templates, Tailwind CSS, Alpine.js
* **Build Tool:** Vite
* **Database:** SQLite (zero-config)
* **Authentication:** Laravel Breeze
