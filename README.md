# Web Technologies Lab Tasks

This repository contains a collection of laboratory assignments completed as part of the Web Technologies course. Each directory represents a specific lab task, progressing from basic front-end structure to dynamic server-side applications.

---

## 📁 Repository Structure

The project is organized into modular folders for easy navigation:

- **Lab1/**: Basic HTML structure and semantics  
- **Lab2/**: Responsive E-commerce landing page  
- **Lab3/**: PHP-based Authentication System  
- **Lab4/**: CRUD Operations with Database Integration  
- **Lab5/**: Student Portal with Dynamic Routing and Video Demo  
- **Lab6/**: Advanced Dynamic Routing & Parameter Handling  
- **Lab7/**: Laravel Employee Management System (CRUD)  
- **Lab8/**: Laravel Advanced Features (Validation & Events)  

---

## 🧪 Lab Details

### Lab 01: Simple HTML Structure

Focuses on the foundational building blocks of the web. This task involves creating a standard document structure using semantic HTML5 tags.

- Key concepts: Headings, paragraphs, lists, and document metadata  

---

### Lab 02: E-commerce Landing Page

A visually appealing landing page for an online store developed using HTML and CSS.

- Features: Navigation bars, product grids, and call-to-action buttons  
- Focus: CSS Flexbox/Grid for layout and responsive design  

---

### Lab 03: Authentication System

A functional backend implementation using PHP to handle user access.

- **Sign-up**: User registration and data persistence  
- **Login**: Credentials verification and session initiation  
- **Dashboard**: Protected route for authenticated users  
- **Logout**: Session destruction and redirection  

---

### Lab 05: Student Portal (Dynamic Parameters)

A portal focusing on data identification through URL parameters and dynamic content fetching.

- Uses GET parameters (e.g., `student_id`)  
- Dynamic editing pages based on ID  

🎥 **Video Demo**:

---

### Lab 06: Advanced Dynamic Routing & Parameter Handling

Builds on the student portal by improving routing logic and handling more complex parameter-based data interactions.

- Cleaner routing structure  
- Improved data fetching  
- Better separation of concerns  

---

### Lab 07: Employee Management System (Laravel CRUD)

A secure web application built with Laravel to manage employee records.

#### 🛠️ Tech Stack
- PHP 8.4+ & Laravel 13  
- Blade Templates & Tailwind CSS (Laravel Breeze)  
- SQLite database  

#### ✨ Features
- Authentication system  
- Full CRUD operations  
- Database migrations  

🎥 **Video Demo**:  
https://drive.google.com/file/d/1bon8t7DMHMEVO5JA9ybCDRYkJSfiMIsz/view?usp=sharing  

---

### Lab 08: Advanced Laravel Features (Validation & Events)

Enhances the Employee Management System with more production-level backend practices.

#### ✨ Features
- Robust server-side validations  
- Laravel Events & Listeners (event-driven architecture)  
- Middleware-based route protection  
- Improved CRUD with validation layers
