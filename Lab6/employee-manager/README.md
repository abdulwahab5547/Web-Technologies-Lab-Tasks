# 🚀 Employee Management System (CRUD)

A streamlined, secure web application built with **Laravel 13** to manage employee records. This project demonstrates core backend proficiency, including authentication, database migrations, and full CRUD functionality.

## 🛠️ Tech Stack & Tools
* **Backend:** PHP 8.4+ & Laravel 13
* **Frontend:** Blade Templates & Tailwind CSS (via Laravel Breeze)
* **Database:** SQLite (for zero-config portability)
* **Environment:** Developed on macOS (Apple Silicon) using Laravel Herd
* **Agentic Workflow:** Built using **Laravel Boost** to optimize AI-assisted development.

---

## ✨ Features
* **Secure Authentication:** User registration and login powered by Laravel Breeze.
* **Full CRUD Operations:** * **Create:** Add new employees with validation (Email, Position, Salary).
    * **Read:** A clean, responsive dashboard listing all staff.
    * **Update:** Dynamic forms to edit existing employee details.
    * **Delete:** Secure removal of records with confirmation.
* **Middleware Protection:** Employee data is only accessible to authenticated users.
